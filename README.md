# Engie API

Api for the Engie Excersise

## Preconditions:

- Python 3

## Clone the project

```
git clone https://github.com/GonzaloJCY/Engie_API.git
```

## Run local

### Install dependencies

```
pip install -r requirements.txt
```

### Run server

```
uvicorn app.main:app --reload
```

### Run test

```
pytest test/test.py
```
